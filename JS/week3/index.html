Object Method, "this"

A function that is a property of an object is called it's method.

Ex : user.age() - user is the object, the method is age.

To access the object, a method can use the "this" keyword and the value of "this" is the object before the dot, the one used to call the method.

Ex. Object method needs to access information stored in the object, like a name

this['refering to user'].name

"this" is evalutaed during the run-time, depending on the context.

A method can be accessed with a dot or ['']

"This" in Javascript is "free", its value is evaluated at call-time and does not depend on where the method was declare, but rather on what the object is before the dot

Arrow functions don't thave their own this

6 Different ways where this can take on new values:

  • in global context
  • in object construction
  • in an object method
  • in a simple function
  • in an arrow function
  • this in an event listener

Ex of this in object construction:

function Human (age) {
this.age = age
}
let greg = new Human(22)
let thomas = new Human(24)
console.log(greg) // this.age = 22
console.log(thomas) // this.age = 24

Namespacing

To prevent naming from becoming too complex, namespacing can be used by placing all our functions inside an object. Then the function must first be called by the object then the property. This helps prevent names from being used or to keep function names more simple and organized

Random Numbers

Math.random() creates random numbers from 0 to 1. To generate a random number between 0 and 10, simply multiply math.random() * 10. To generate an random integer, you can use math.floor around the multiplication.

Regular Expressions

Syntax for creating a new RegExp: const pattern = new RegExp('[a-zA-Z]+ing');

A sequence of characters can be represented using a " - "
ex: /[A-Z]/
Digits: ex: /[0-9]/
^ negates the sequences: /[^A-Z]/ this means any characters that is not a capital

Regular Expressions are objects and have the following properties:

  • Global Property: makes the pattern return all matches, by default only looks for the first occurrence of a match
  • ignoreCase makes the pattern case-insensitie, default is case sensitive
  • multiline- makes the pattern multipline, by default it stops at the end of a line

Document Object Model

Tree-like structure of connected nodes in an HTML document. DOM is not actually part of JavaScript, it is language agnostic, but in Javascript we can access this part of the web by using the built in object called document.

The DOM stores whitespace that is in the html document as text nodes as well.

DOM Methods

.nodeType tells what type of node it is:

  • Result 1: element
  • Result 2: attribute
  • Result 3: text
  • Result 8: comment
  • Result 9: body

.nodeName - finds the name of the element

Legacy DOM Shortcut Methods

Document.body, Document.images, Document.links, Document.anchors, Document.forms.

Other properties: document.images[0], has .length
